# Proveedores de APIs

## Activos en la app

- Paises: `https://restcountries.com/v3.1/all?fields=name,capital,region,population,area,flags,currencies,cca2,cca3,latlng`
- Clima: `https://api.open-meteo.com/v1/forecast?latitude=40.4&longitude=-3.7&current=temperature_2m,apparent_temperature,relative_humidity_2m,weather_code`
- Divisa: `https://api.frankfurter.app/latest?from=EUR&to=USD`
- Economia: `https://api.worldbank.org/v2/country/ES/indicator/NY.GDP.MKTP.CD?format=json&mrnev=1`
- Cultura: `https://en.wikipedia.org/api/rest_v1/page/summary/Spain`

- REST Countries: sin auth
- Open-Meteo: sin auth
